# Audio Processing Pipeline Refactoring Plan

## Summary

Refactor the core audio processing loop to add a size threshold for direct upload, implement smarter file size cascading, speed up processing with PyAV, and improve intermediate file management.

---

## Requirements

1. **New `--size-threshold` param** (default 100MB) - below = direct upload if API accepts format
2. **Smarter size algorithm** - check extracted audio size before further compression
3. **Speed improvements** - replace ffmpeg subprocess with PyAV (~2-3x faster)
4. **Intermediate file management** - consistent naming, only delete after successful JSON
5. **Chunking speedup** - streaming instead of full file load

---

## Current State

### File: `audio_transcribe/transcribe_helpers/audio_processing.py`

**Current `optimize_audio_for_api()` flow (lines 505-656):**
```
1. Check if original <= API limit → return original
2. If video → extract audio (ffmpeg subprocess)
3. Convert to FLAC (ffmpeg subprocess)
4. Convert to MP3 128k (ffmpeg subprocess)
5. Return best result (cleanup intermediates immediately)
```

**Issues:**
- No passthrough for small files that don't need processing
- Always converts even when API accepts original format
- Each step spawns new ffmpeg subprocess (~50-200ms overhead each)
- Intermediates deleted immediately (lost on error)
- Uses tempfile for FLAC/MP3 (random names, hard to reuse)

### Phase 1: IntermediateFileManager Class

**New file:** `audio_transcribe/transcribe_helpers/intermediate_files.py`

```python
from pathlib import Path
from dataclasses import dataclass
from enum import Enum
from typing import Dict, Optional
from loguru import logger

class FileOperation(Enum):
    EXTRACTED = "extracted"
    CONVERTED_FLAC = "flac"
    CONVERTED_MP3 = "mp3"
    CHUNK = "chunk"

@dataclass
class IntermediateFile:
    path: Path
    operation: FileOperation
    source_path: Path
    size_mb: float = 0.0

class IntermediateFileManager:
    """
    Manages intermediate files with:
    - Consistent naming: {stem}_intermediate_{operation}.{ext}
    - Temp directory alongside original file
    - Deferred cleanup (only after success)
    """

    def __init__(self, original_path: Path):
        self.original_path = Path(original_path)
        self.temp_dir = self.original_path.parent / f".transcribe_temp_{self.original_path.stem}"
        self._files: Dict[FileOperation, IntermediateFile] = {}
        self._preserve = False

    def setup(self) -> None:
        self.temp_dir.mkdir(parents=True, exist_ok=True)

    def get_path_for(self, operation: FileOperation, ext: str) -> Path:
        return self.temp_dir / f"{self.original_path.stem}_intermediate_{operation.value}.{ext}"

    def register(self, path: Path, operation: FileOperation, source: Path) -> IntermediateFile:
        f = IntermediateFile(path, operation, source, path.stat().st_size / 1024 / 1024)
        self._files[operation] = f
        return f

    def preserve_on_error(self) -> None:
        self._preserve = True
        logger.info(f"Preserving intermediates in: {self.temp_dir}")

    def cleanup(self) -> None:
        if self._preserve:
            return
        for f in self._files.values():
            if f.path.exists():
                f.path.unlink()
        if self.temp_dir.exists() and not any(self.temp_dir.iterdir()):
            self.temp_dir.rmdir()
```

### Phase 2: PyAV Backend

**New file:** `audio_transcribe/transcribe_helpers/pyav_backend.py`

```python
from pathlib import Path
from typing import Optional
from loguru import logger

PYAV_AVAILABLE = False
try:
    import av
    PYAV_AVAILABLE = True
except ImportError:
    pass

def is_pyav_available() -> bool:
    return PYAV_AVAILABLE

def get_duration_seconds(audio_path: Path) -> float:
    """Get duration without subprocess (10x faster than ffprobe)."""
    if not PYAV_AVAILABLE:
        return _get_duration_ffprobe(audio_path)
    try:
        with av.open(str(audio_path)) as container:
            return container.duration / av.time_base if container.duration else 0.0
    except:
        return _get_duration_ffprobe(audio_path)

def extract_audio_pyav(input_path: Path, output_path: Path) -> bool:
    """Stream copy audio from video (2-3x faster than subprocess)."""
    if not PYAV_AVAILABLE:
        return False
    try:
        with av.open(str(input_path)) as inp:
            audio_stream = inp.streams.audio[0]
            with av.open(str(output_path), 'w') as out:
                out_stream = out.add_stream(template=audio_stream)
                for packet in inp.demux(audio_stream):
                    if packet.dts is None:
                        continue
                    packet.stream = out_stream
                    out.mux(packet)
        return True
    except Exception as e:
        logger.warning(f"PyAV extraction failed: {e}")
        return False

def convert_to_flac_pyav(input_path: Path, output_path: Path) -> bool:
    """Convert to FLAC 16kHz mono using PyAV."""
    if not PYAV_AVAILABLE:
        return False
    try:
        with av.open(str(input_path)) as inp:
            audio_stream = inp.streams.audio[0]
            with av.open(str(output_path), 'w') as out:
                out_stream = out.add_stream('flac', rate=16000)
                out_stream.layout = 'mono'
                resampler = av.AudioResampler(format=out_stream.format, layout='mono', rate=16000)
                for frame in inp.decode(audio_stream):
                    for resampled in resampler.resample(frame):
                        for packet in out_stream.encode(resampled):
                            out.mux(packet)
                for packet in out_stream.encode():
                    out.mux(packet)
        return True
    except Exception as e:
        logger.warning(f"PyAV FLAC conversion failed: {e}")
        return False
```

```python
def can_passthrough(file_path: Path, api_name: str, threshold_mb: float) -> bool:
    """Check if file can skip processing entirely."""
    size_mb = file_path.stat().st_size / (1024 * 1024)
    api_limit = get_api_file_size_limit(api_name)

    if size_mb > min(threshold_mb, api_limit):
        return False

    reqs = API_FORMAT_REQUIREMENTS.get(api_name, {"accepts_video": True})
    ext = file_path.suffix.lower()

    # Video files can only passthrough to APIs that accept them
    if ext in ['.mp4', '.mkv', '.avi', '.mov', '.webm']:
        return reqs.get("accepts_video", False)

    # FLAC-required APIs need FLAC format
    if reqs.get("requires_flac") and ext != '.flac':
        return False

    return True
```

**Refactored `optimize_audio_for_api()` signature:**
```python
def optimize_audio_for_api(
    input_path: Union[str, Path],
    api_name: str,
    size_threshold_mb: float = 100.0,
    preserve_on_error: bool = True
) -> OptimizationResult:
```

**New algorithm:**
```
1. Check passthrough (new):
   - size < threshold AND size < API limit AND format accepted?
   → Return original immediately

2. Extract audio (if video):
   - Use PyAV (fast) with ffmpeg fallback
   - Check if extracted fits API limit
   - If API accepts any format AND extracted fits → return (skip FLAC)

3. Convert to FLAC (if needed):
   - Only if API requires FLAC OR file still too large
   - Use PyAV with ffmpeg fallback

4. Convert to MP3 (last resort):
   - Only if FLAC still exceeds limit

5. Return with IntermediateFileManager attached
   - Cleanup deferred until success
   - On error: preserve intermediates
```

**Update OptimizationResult:**
```python
@dataclass
class OptimizationResult:
    path: Path
    is_temporary: bool
    size_mb: float
    bytes_per_second: float
    intermediate_manager: Optional['IntermediateFileManager'] = None

    def cleanup(self) -> None:
        if self.intermediate_manager:
            self.intermediate_manager.cleanup()
```

## Risks & Mitigations

1. **PyAV installation issues on Windows**
   - Mitigation: Graceful fallback to ffmpeg subprocess
   - Document: "Install with `uv add av` or use fallback mode"

2. **Intermediate files filling disk**
   - Mitigation: Temp dir created per-file, cleaned on success
   - User can manually delete `.transcribe_temp_*` dirs

3. **Breaking existing workflows**
   - Mitigation: All defaults preserve current behavior
   - `--size-threshold 0` forces processing (old behavior)
